---
title: Determinar el Tiempo de Ejecución de un Comando de Shell
description: Aprende cómo rastrear el tiempo consumido por un comando de shell en un sistema basado en Unix.
date: 2024-02-08
---

¿Alguna vez te has preguntado cuánto tiempo tomó ejecutar tu comando de shell? Aquí tienes un truco sencillo para averiguarlo.

```shell
time tu-comando
```

Este comando ejecutará 'tu-comando' y luego mostrará información sobre los recursos utilizados por él, incluido el tiempo que tomó ejecutarlo. Es una excelente manera de optimizar tus scripts.
