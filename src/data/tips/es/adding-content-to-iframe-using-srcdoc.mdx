---
title: Agregar contenido a un iframe utilizando srcdoc
description: Aprenda cómo agregar contenido a un iframe directamente utilizando el atributo srcdoc en HTML.
date: 2024-02-08
---

Utilizar el atributo srcdoc en un iframe le permite agregar contenido HTML directamente dentro de él.

```html
<iframe srcdoc="<p>Hola, Mundo!</p>"></iframe>
```

El atributo srcdoc le permite escribir un documento HTML directamente dentro de un iframe, sin necesidad de llamar a un archivo externo. Proporciona una forma fácil y flexible de administrar el contenido de un iframe.
